# Tiny Data Centers — Span XFRA strict analysis

An interactive presentation and supporting research analyzing Span's XFRA — a
distributed AI data center system that places GPU compute pods on the side of
individual homes — and exploring what an XFRA-style product would look like
for Toledo and Northwest Ohio.

**Live presentation:** https://vanderpoolteacher.github.io/tiny-data-centers/

## What's in this repo

- **[`index.html`](./index.html)** — the interactive presentation (17 slides
  of strict analysis + concept work on Northwest Ohio). Open in any modern
  browser. Navigate with arrow keys, space, or the controls at the bottom.
- **[`xfra-analysis-report.md`](./xfra-analysis-report.md)** — the strict
  analytical report the deck is built from: technical viability, business
  viability, strategic positioning, risk inventory, adjacencies.
- **[`span-xfra-research-brief.md`](./span-xfra-research-brief.md)** — primer
  on what Span XFRA actually is: hardware spec, power envelope, deployment
  model, economics.
- **[`span-xfra-competitive-landscape.md`](./span-xfra-competitive-landscape.md)** —
  side-by-side analysis of the competitive set: direct peers, modular
  prefab, telco edge, hyperscaler edge, behind-the-meter generation,
  grid-flex orchestration, decentralized GPU networks.
- **[`concept-doc.md`](./concept-doc.md)** — working draft exploring what a
  Span/XFRA alternative might look like across four directional bets.
